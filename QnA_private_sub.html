난향수정
Makefile은 "make" 명령어를 사용하여 소스 코드를 컴파일하고 실행하기 위한 규칙을 정의하는 것 입니다.

현재 주어진 상황에서는 이미 최신의 실행 파일이 생성되어 있어서 더 이상 변경할 필요가 없기 때문에 "make: Nothing to be done for 'all'."이라는 메시지가 나타나게 됩니다.

"make all" 명령을 여러 번 실행하더라도 동일한 메시지가 나타날 것입니다.

“make all” 명령을 실행하고 싶으면, "make clean" 명령을 사용하여 불필요한 파일을 제거하면 됩니다.
